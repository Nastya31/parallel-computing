# Лабораторная работа №1 Вычисление факториала с помощью многопоточности
В данной пргограмме реализован алгоритм вычисления факториала с помощью двух потоков.
В первую очередь программа выполняет проверку на четность введенного числа. Если число четное, то второй поток начнет расчет со значения равном этому числу, деленное на 2. Иначе, с этого же числа + 1. Вычисления производятся в функции void factorial
